# My-Gallery

​
List of available endpoints:
​
- `POST /register`
- `POST /login`
- `POST /googleLogin`
- `GET /tasks`
- `POST /tasks`
- `GET /tasks/:id`
- `PUT /tasks/:id`
- `DELETE /tasks/:id`




### POST /register

description: 
  register user

Request:

- data:

```json
{
  "firstName":"string",
  "lastName":"string",
  "email": "string",
  "password": "string"
}
```

Response:

- status: 201
- body:
  ​

```json
{
  "id": "integer",
  "fullname":"string",
  "email": "string"
}
```

- status: 400
- body:
  ​

```json
{
  "message": [
    "email must be unique",
    "email musn't empty",
    "first Name musn't empty",
    "last Name musn't empty",
    "password musn't empty",
    "email musn't null",
    "first Name musn't null",
    "last Name musn't null",
    "password musn't null"
  ]
}
```

- status: 500
- body:
  ​

```json
{
  "message": "Internal server error"
}
```
